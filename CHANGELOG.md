# Changelog

All notable changes to Hera are documented here. Versioning is semantic-ish: MAJOR.MINOR.PATCH.

## [2.7.0] - 2026-07

Current public release. Feature set at time of open-sourcing:

- Equirectangular to cubemap face extraction (105° face FOV) and equirectangular frame extraction
- Indoor/Outdoor face-drop presets
- COLMAP cameras.txt auto-generation (pinhole intrinsics) and sequential-matching pair-list export
- AprilTag (tag36h11) sheet generator
- Capture planner: multi-room scene builder, rectangular/circular rooms, drag-and-drop snapping, doorway junction detection
- Rig profile save/load (JSON), scene save/load (JSON)
- Workflow tips window
- Optional tkinterdnd2 drag-and-drop with stdlib fallback
